# 🇬🇧 English

A lightweight and fully local **product label generator** built with pure **HTML, CSS, and JavaScript**.

Create clean, print-ready labels directly in your browser with **no installation, no backend, and no external libraries**.

Perfect for warehouses, manufacturing, packaging teams, and local inventory workflows.

## Features

✅ Fully local — works offline
✅ No installation required
✅ Instant label preview
✅ Built-in barcode generation (**Code128**)
✅ Custom label size support
✅ Print-ready layout
✅ Save labels as **PDF** via browser print dialog
✅ Zero dependencies

## Included Fields

* Product Name
* Packaging Date
* Weight
* Barcode
* Item Number

## How It Works

1. Open the application in your browser
2. Fill in product information
3. Preview the generated label instantly
4. Click **Print / Save as PDF**
5. Print the label or save it as PDF

## Technologies Used

* HTML5
* CSS3
* Vanilla JavaScript
* SVG Barcode Rendering

No frameworks. No APIs. No tracking.

# 🇷🇺 Русский

Лёгкий и полностью локальный **генератор товарных этикеток**, созданный на чистом **HTML, CSS и JavaScript**.

Позволяет быстро создавать готовые к печати этикетки прямо в браузере — **без установки, сервера и сторонних библиотек**.

Подходит для складов, производства, упаковки товаров и локального учёта.

## Возможности

✅ Полностью локальный — работает офлайн
✅ Не требует установки
✅ Мгновенный предпросмотр этикетки
✅ Встроенная генерация штрихкодов (**Code128**)
✅ Поддержка кастомных размеров этикеток
✅ Готовая вёрстка для печати
✅ Сохранение в **PDF** через системную печать браузера
✅ Без зависимостей

## Поля этикетки

* Название товара
* Дата упаковки
* Вес
* Штрихкод
* Номер товара / заказа

## Как использовать

1. Откройте приложение в браузере
2. Заполните информацию о товаре
3. Мгновенно просмотрите результат
4. Нажмите **«Печать / сохранить в PDF»**
5. Распечатайте этикетку или сохраните её в PDF

## Используемые технологии

* HTML5
* CSS3
* Чистый JavaScript (Vanilla JS)
* SVG-генерация штрихкодов

Без фреймворков. Без API. Без отслеживания.
